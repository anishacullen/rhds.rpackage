The goal of rhds.rpackage is to execute functions required for the Reproducible Health Data Science Course

## Installation

You can install the development version of rhds.rpackage from [GitHub](https://github.com/) with:

``` r
# install.packages("pak")
pak::pak("anishacullen/rhds.rpackage")
```

## Example

This is a basic example which shows you how to solve a common problem:

``` r
library(rhds.rpackage)
## basic example code
```
